## EqualizeHist

[Histogram Equalization](https://en.wikipedia.org/wiki/Histogram_equalization) for Vapoursynth.

### Usage
```python
ehist.EqualizeHist(vnode clip)
```
### Parameters:

- clip\
    A clip to process. 8-bit integer only.
